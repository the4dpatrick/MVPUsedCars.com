MVPUsedCars.com
---------------
![MVP Logo](https://raw.github.com/the4dpatrick/mvpusedcars.com/master/app/assets/images/logo.jpg "MVP Used Cars logo")

A website I made for my family's business. The website has both a visitor and admin side. The visitor side uses information defined in the database through the admin's dashboard.

Image hosting uses the Paperclip gem and AWS S3.
Font-icons from entypo.com
CSS Framework: Foundation Zurb
CSS compiled with Sass
